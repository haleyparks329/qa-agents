Start here for shared context:

1. Read `agents/shared/core-rules.md`.
2. Resolve the active profile with `python3 profile.py agent-context <agent>`.
3. Load any relevant KB state with `python3 kb.py stats` and `python3 kb.py query <surface>`.

The profile-specific `house-rules.md` may add generic app-category constraints. Agent specs should stay focused on role, workflow, and handoff behavior.
