# Little Bytes Profile Rules

Little Bytes is a future deterministic bakery reference app for exercising QA Agents evidence loops. Treat it as a controlled QA laboratory, not a product showcase.

Use only committed seed data. Prioritize pricing accuracy, coupon behavior, checkout completion, cart persistence, inventory availability, and accessible checkout controls.

This profile must remain configurable across machines. Do not add private paths, credentials, production URLs, or real customer data.
